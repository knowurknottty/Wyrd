## Corporate Euphemisms

| Euphemism | Translation | WYRD Analysis |
|-----------|-------------|---------------|
| "Rightsizing" | Firing people | Making the 'right' size = removing humans. The "right" masks violence with virtue. |
| "Synergy" | Exploitation | Greek *syn-* (together) + *ergon* (work)—extracting more work through togetherness |
| "Leverage" | Using without reciprocity | Old French *levier* (to raise)—raising yourself by standing on others |
| "Human resources" | Commodified people | Humans as raw material to be extracted. "Resources" = things to use up. |
| "Stakeholder value" | Profit for owners | Those who hold stakes (as in vampires). Value for holders, not creators. |
| "Optimization" | Stripping away humanity | Making you into an optimal machine. Optimize = make efficient, regardless of cost. |
| "Best practices" | Conformity | Doing what everyone else does. "Best" = most common, not most effective. |
| "Thought leadership" | Status signaling | Not thinking—leading thoughts (controlling the narrative). |
| "Pivot" | Giving up | Basketball term for changing direction. Sounds athletic, not desperate. |
| "Sunset" | Kill/discontinue | Sunsets are beautiful. Discontinuation is not. |
| "Offline" | Excluding someone | "Let's take this offline" = let's exclude you from this conversation. |
| "Align" | Make you agree | Not finding truth—making vectors point same direction. |
| "Deliverables" | Your soul | What can be delivered = commodified output. |
| "Bandwidth" | Energy/attention | Reducing human capacity to internet connection. |
| "Circle back" | Avoid commitment | Promise to return, rarely do. |
| "Move the needle" | Show any result | Even tiny shifts count as progress. |
| "Low-hanging fruit" | Easy victims | Fruit = passive, waiting to be picked. |
| "Boil the ocean" | Attempt the impossible | Making failure sound like effort. |
| "Open the kimono" | Reveal secrets | Sexualized metaphor for business disclosure. |
| "Drinking the Kool-Aid" | Believing propaganda | References Jonestown massacre—deadly belief. |

## Political Euphemisms

| Euphemism | Translation | WYRD Analysis |
|-----------|-------------|---------------|
| "Collateral damage" | Killing civilians | Damage to property, not murder of people. Collateral = beside the main point. |
| "Enhanced interrogation" | Torture | Making questioning 'better.' Enhanced = improved. |
| "Quantitative easing" | Printing money | Easing the quantity of money = inflation. Sounds technical, not theft. |
| "Austerity measures" | Cutting services | *Austērus* (harsh)—making severe, stern. Harshness as virtue. |
| "National security" | State secrecy | Security of the state, not the people. Whose nation? Whose security? |
| "Regime change" | Overthrowing governments | *Regime* from Latin *regimen* (rule)—changing who rules. |
| "Ethnic cleansing" | Genocide | Cleansing = purifying. Murder as hygiene. |
| "Friendly fire" | Killing allies | Fire = violent. Friendly = non-violent. Contradiction masking tragedy. |
| "Shock and awe" | Terror bombing | Shock = trauma. Awe = reverence. Terror as religious experience. |
| "Boots on the ground" | Sending soldiers | Boots = equipment, not humans. Ground = foreign soil we don't belong on. |
| "Surgical strike" | Explosive attack | Surgery = precise healing. Bombing = destructive. |
| "Coercive diplomacy" | Threats | Diplomacy = negotiation. Coercive = forced. Oxymoron. |
| "Extraordinary rendition" | Kidnapping | Extraordinary = beyond ordinary. Rendition = handing over. |
| "Preventive war" | Aggression | Starting war to prevent war. War is peace. |
| "Freedom fighters" | Rebels we like | Same actions, different labels based on alignment. |
| "Terrorists" | Rebels we don't like | Fear-based label to dehumanize opposition. |
| "Economic sanctions" | Starving populations | Sanction = permission. Actually, restriction that causes suffering. |
| "Targeted killing" | Assassination | Targeted = precise. Killing = murder. |
| "Persons of interest" | Suspects | Interest = curiosity. Actually, suspicion of crime. |
| "Enhanced pat-down" | Groping | Enhanced again. Pat-down = casual. Actually invasive search. |

## How to Use This Dictionary

### Personal Practice
1. When you hear a euphemism, pause
2. Ask: What is actually being said?
3. Check the WYRD Analysis
4. Respond to reality, not the mask

### Social Situations
- Point out euphemisms gently: "When you say 'rightsizing,' do you mean firing people?"
- Reframe in plain language: "Let's call it what it is"
- Notice who benefits from the euphemism

### The Liberation Angle

Every euphemism has a function:
- **Distance** the speaker from the act
- **Soften** harsh realities
- **Obscure** power relationships
- **Normalize** the unacceptable

**The WYRD response:** Name it precisely. Precision is power.
